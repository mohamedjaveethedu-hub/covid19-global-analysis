# COVID-19 Global Data Analysis

## Project Overview

This project analyzes global COVID-19 data using:

- Python
- Pandas
- SQL
- Matplotlib
- Seaborn

The project includes:
- Data cleaning
- SQL analysis
- Exploratory Data Analysis (EDA)
- Data visualization

---

## Key Insights

1. Identified countries with highest COVID cases
2. Analyzed countries with highest deaths
3. Visualized monthly global case trends
